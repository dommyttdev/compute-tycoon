# Triage labels

Apply at most one workflow-state label to an issue. Type labels such as `bug`,
`enhancement`, and `documentation` may be combined with it.

| Role | GitHub label | Meaning |
| --- | --- | --- |
| Needs triage | `needs-triage` | Maintainer evaluation required |
| Needs info | `needs-info` | Waiting for the reporter |
| Agent ready | `ready-for-agent` | Fully specified for autonomous work |
| Human ready | `ready-for-human` | Human implementation or judgment required |
| Rejected | `wontfix` | Will not be actioned |
