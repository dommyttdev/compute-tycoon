# ADR-0004: 状態バージョン監視による自動保存

- 状態: 採用
- 日付: 2026-07-11

## 文脈

各CLIコマンドが直接JSONを書き込むと、操作応答がI/Oへ依存し、保存処理が各所へ重複する。

## 決定

保存対象の変更時に`state_version`を増やし、別スレッドの`AutosaveWorker`が変化を検出して
Snapshotを保存する。JSONは一時ファイルへ書いた後に置換する。

## 結果

コマンド処理とI/Oが分離され、連続変更をまとめて保存できる。新しい変更操作でversion更新を
忘れると保存されないため、状態変更メソッドはゲーム集約内へ集約する。
